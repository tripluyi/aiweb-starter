# aiweb-starter
这是一个关于 AI 接口调用仓库，更多介绍可以查看 [Wiki](https://ctrip.wiki/)

## 开始

安装
```bash
npm i
```

启动
```bash
npm run dev
```


## 使用
该项目中，每一个例子都可以独立运行，代码互不影响。

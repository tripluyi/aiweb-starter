# aiweb-starter

## 开始

安装
```bash
npm i
```

启动
```bash
npm run dev
```


## 使用
该项目中，每一个例子都可以独立运行，代码互不影响。

### function calling
http://localhost:3000/functioncalling

使用介绍：[OpenAI Function Calling的使用](https://aiweb.viku.org/practice/openai-function-calling-de-shi-yong)

### Server-sent Event
http://localhost:3000/sse
